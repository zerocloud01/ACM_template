## 背包
### 01背包
#### 表示容量，存储价值
```cpp
for(int i=1;i<=m;++i)// 层数
{
    for(int j=t;j>=t[i];--j)// 容量
    {
        dp[j] = max(dp[j],dp[j-t[i]]+v[i]);
    }
}
```
#### 表示价值，存储容量
```cpp
for(int i=1;i<=m;++i)// 层数
{
    for(int j=V;j>=t[i];--j)// 价值
    {
        if(dp[j-v[i]] + w[i] <= t)	dp[j] = min(dp[j],dp[j-v[i]]+w[i]);
    }
}
```
### 完全背包
```cpp
for(int i=1;i<=m;++i)// 层数
{
    for(int j=t[i];j<=t;++j)// 容量
    {
        dp[j] = max(dp[j],dp[j-t[i]]+v[i]);
    }
}
```

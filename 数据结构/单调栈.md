### 单调栈
栈内元素始终单调（栈顶 **更** 满足要求）
> 如更大，更小
```cpp
vector<int> stp;
stack<int> st;
for(int i=1;i<=n;++i)
{
	while(st.size() && check(a[i]))	st.pop();
	stp.push_back(st.size() ? st.top() : -1);
	st.push(a[i]);
}
```
#### 左侧第一个比自己小/大元素
```cpp
// 左侧小
vector<int> ans;
stack<int> st;
for(int i=1;i<=n;++i)
{
	while(st.size() && st.top() >= a[i])	st.pop();
	ans.push_back(st.size() ? st.top() : -1);
	st.push(a[i]);
}
// 左侧大
for(int i=1;i<=n;++i)
{
	while(st.size() && st.top() <= a[i])	st.pop();
	ans.push_back(st.size() ? st.top() : -1);
	st.push(a[i]);
}
```

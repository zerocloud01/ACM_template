## 快速幂
### $mod$ 自变量
```cpp
int qpow(int a,int b,int p) // 快速幂
{
	int z = 1;
	a %= p;
	while(b)
	{
		if(b&1)	z = (z*a)%p;
		a = (a*a)%p;
		b >>= 1;
	}
	return z;
}
```

|子序列|子串|

### LCS - 最长公共子序列
求长度正反皆可，但如果需要还原，或许反向更好，因为需要反向溯源求串。
```cpp
// 求长度部分
int dp[N][N];
for(int i=L1;i>=1;--i)
{
    for(int j=L2;j>=1;--j)
    {
        // 如果两个字符相同，则可以尝试让串长度 +1
        if(s1[i] == s2[j])	dp[i][j] = dp[i+1][j+1] + 1;
        // 不同则继承上一个串
        dp[i][j] = max({dp[i][j],dp[i+1][j],dp[i][j+1]});
    }
}
// 反向溯源
int i=1,j=1;
string ans;
while(i <= L1 && j <= L2)
{
    // 如果两个字符相同，则加入答案
    if(s1[i] == s2[j])
    {
        ans += s1[i];
        i ++, j ++;
    }
    // 否则回归较大的数
    else (dp[i+1][j] > dp[i][j+1] ? i ++ : j ++);
}
```
